Should first create the appropriate main.spec file

Command to transform the Python app into a Windows executable (.exe): pyinstaller main.spec

The output will be put in the dist folder (it will ask me to remove all the previous content inside the dist folder)

Copy all images used in my app from the repo into dist/main (e.g. logo.png)

Finally, use inno setup compiler to create an installation wizard that will be used to install the created executable to other Windows devices
